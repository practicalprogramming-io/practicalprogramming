practicalprogramming.io is for 

Tutorials and resources about the craft of programming and problem solving with computers in today's industry - written by professional software engineers!

Our mission at practicalprogramming.io is to teach people with a thrist for knowledge!  Here you'll find 'how-to' content about how to work with different programming languages, frameworks and database technologies across the whole software stack.  We expect users to understand at least one programming language at a beginner level in order to understand and follow along with tutorials (head over to resources for some excellent third-party content if you are not yet comfortable programming at a beginner level).
As advocates of free and open source software, every single tutorial available on practicalprogramming.io has a public repository with working examples hosted on GitHub for you to clone and work along with.  Our seasoned engineers have advanced level understanding of the inner workings and best practices of Python 2.7 and 3.5, JavaScript ES5 and ES6, stable and bleeding edge Node.js, PostgreSQL and MongoDB.  They'll teach you how to develop with these tools across the software stack for production deployment on Linux servers using tools like Nginx, Apache, Tomcat and a variety of Unix command line utilities.
